**출처**: https://www.contestkorea.com | 현재 진행 공모전 2,740건+

- 수상작 갤러리 — 수상 레퍼런스 탐색 가능

### 약점
- UI 노후화 (2000년대 초반 게시판 구조)
- 달력 필터 빈약 — 카테고리별 색상 구분 없음
- 해커톤·예창패·정부지원금 섹션 없음
- 모바일 최적화 미흡 (구식 레이아웃)
- 검색 URL 파라미터 기반 — 복잡한 필터 조합 불편
- 북마크/관심 공고 저장 기능 없음 (로그인 시 일부 제공)
- 알림 기능 없음

### OpportuMap에 없는 기능

- 공모전 뉴스/보도 섹션
- 인기순·조회순 정렬

## 2. 링커리어 (linkareer.com)

**출처**: https://linkareer.com/list/contest | https://community.linkareer.com

- **세분화된 필터**: 공모분야 / 시상규모 / 수상혜택 / 응모대상 / 주최기관 / 주최기간
- **스크랩 기능** — 관심 공고 저장 및 목록화
- 공모전 + 대외활동 + 인턴 + 동아리 통합
- 팀원 모집 게시판 (해커톤·공모전 팀 구성)
- 활동 일지/후기 커뮤니티 (실제 참가자 경험 공유)
- 취업/스펙 콘텐츠와 연계 (공모전 → 스펙 관리 흐름)

### 약점
- 캘린더 뷰 없음 — 리스트 형식만 제공
- 정부지원금·예창패 완전 미포함

- 마감일 알림 기능 없음 (스크랩만 있고 알림 미연결)
- 정보 밀도 낮음 — 상세 요건보다 카드 요약 위주

### OpportuMap에 없는 기능
- **스크랩(북마크)** + 스크랩 목록 페이지
- **팀원 모집** 게시판
- 공모전 후기/활동 일지 커뮤니티
- 시상규모 필터 (상금 크기 기준 탐색)
- 수상혜택 필터 (인턴, 장학금, 해외연수 등)
- 취업 연계 콘텐츠

**출처**: https://www.k-startup.go.kr | 중소벤처기업부 공식 포털

- **캘린더 뷰 존재** — "창업자를 위한 Schedule" 달력에서 사업공고 시각화
- **맞춤형 검색**: 창업단계 / 지역 / 분야 / 사업유형 4개 필터
- 공고 신청 내역 마이페이지 (로그인 시)
- 8개 지원 카테고리 체계적 분류 (창업교육/시설·공간/멘토링/사업화/R&D/글로벌/행사·네트워크/융자)
- 통합 로그인 (중소벤처기업부 전 사이트 연동)

### 약점
- 공모전·해커톤 없음 — 창업 지원사업 전용
- 검색 UX 열악 (정부 포털 특유의 복잡한 UI)
- 캘린더 실용성 낮음 — 공고 수가 너무 많아 가독성 저하
- 모바일 UX 미흡
- 알림 기능 없음 (외부 서비스 "창업알리미"로 별도 보완)
- 온보딩 없음 — 처음 방문자에게 복잡하게 느껴짐

### OpportuMap에 없는 기능
- **업력/창업단계 필터** (예비창업자 / 3년 이하 / 7년 이하 등)
- **사업유형 필터** (사업화 / R&D / 글로벌 / 융자 등)

- 중소기업 지원 연계 (기업마당, 소상공인마당 등)

## OpportuMap 개선 우선순위 (상위 10개)

> 기준: 경쟁사 대비 부재 기능 + 사용자 페인포인트 직결 + 구현 난이도 고려

### P0 — 즉시 (핵심 차별화 완성)

**1. D-day 카운트다운 카드 표시**
- 근거: Devpost의 가장 강력한 UX 요소. "11 days left" 방식으로 마감 긴박감 전달
- 현황: 마감일 텍스트 있으나 D-day 계산값 미표시
- 효과: 사용자의 마감일 인지 → 즉각 행동 유발

**2. 북마크/관심 공고 저장 기능**
- 근거: 링커리어 스크랩 기능이 핵심 재방문 동기. 현재 OpportuMap 완전 부재
- 현황: 없음
- 효과: 재방문율 증가, 세션당 체류 시간 증가

**3. 시상규모/상금 필터 + 카드 강조**
- 근거: 링커리어 필터 항목 중 "시상규모" 인기 1위. Devpost 상금 표시가 즉각적 관심 유발
- 현황: 상금 정보 없음
- 효과: 고상금 공모전 탐색 니즈 충족

### P1 — 단기 (UX 완성도)

**4. 마감 알림 (웹 푸시 / 이메일)**
- 근거: PRD에서 P2로 분류했으나, 경쟁사 모두 미지원 → **차별화 1순위**. 창업알리미가 K-Startup의 알림 부재를 별도 서비스로 채우며 사용자 흡수 중
- 현황: 없음
- 효과: 가장 강력한 재방문 트리거. "마감일 놓침" 핵심 페인포인트 해결

**5. 분야 카테고리 세분화**
- 근거: 위비티 10개 vs 현재 4개 (공모전/해커톤/예창패/지원금). 공모전 내부에서도 "디자인", "IT/개발", "영상" 등 세분 필요
- 현황: 상위 4개 카테고리만 존재
- 효과: 타겟 분야 탐색 정확도 향상

**6. 참가자 수 / 경쟁률 표시**
- 근거: Devpost의 "2,048 participants" 표시가 사회적 증거로 작용 → 인기 공모전 식별
- 현황: 없음
- 효과: 인기도 기반 탐색 가능, 참여 동기 부여

### P2 — 중기 (차별화 기능)

**7. 수상 요건 체크리스트 강화**
- 근거: 현재 OpportuMap 상세 모달에 체크리스트 있으나 내용이 일반적. Devpost 수상작 아카이브처럼 "이런 팀이 수상했다" 레퍼런스 연결 필요
- 현황: 기본 체크리스트만 존재

**8. 팀원 모집 / 팀 구성 기능**
- 근거: 링커리어와 Devpost 모두 제공. 해커톤/공모전에서 팀 구성이 최대 진입 장벽
- 현황: 없음

**9. 수상작 갤러리 / 레퍼런스 아카이브**
- 근거: 콘테스트코리아와 Devpost 모두 제공. "어떻게 수상했나"에 대한 참고 자료 수요 큼
- 현황: 없음

**10. 메일링 뉴스레터 (주간 맞춤 공고 발송)**
- 근거: 위비티 메일링 서비스가 재방문 없이도 공고를 소화시키는 강력한 리텐션 도구. Devpost 주간 뉴스레터 동일
- 현황: 없음
