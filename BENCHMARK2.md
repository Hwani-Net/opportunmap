## 1. 링커리어 (linkareer.com)

### 핵심 기능
- 공모전/대외활동/인턴/채용 통합 플랫폼 (공모전에만 국한하지 않음)
- 월 150만명 이상 이용 (국내 최대 규모)
- 모바일 앱 제공 (iOS/Android)
- CBT 무료 시험 플랫폼 연계
- 자기소개서 관리 기능

### UI/UX 특징
- 상단 메뉴: 신입/인턴, 대외활동, CBT, 커뮤니티, 자기소개서, 멘토, 채널
- "금주의 인기공고", "실시간 인기 채용" 섹션 구성
- 카드형 리스트 레이아웃 (썸네일 + 제목 + 마감일)
- 취업 플랫폼 성격이 강해 공모전은 서브 카테고리

### 필터/검색/알림
- 5중 필터: 공모분야 / 시상규모 / 수상혜택 / 응모대상 / 주최기관
- **공고 달력**: 월간/주간 뷰 제공, 시작일+마감일 동시 표시 (핵심 차별화)
- **내 스크랩 달력**: 스크랩한 공모전만 모아 보는 개인 캘린더
- D-1 마감 알림 (앱 푸시)
- 최신순 / 마감순 / 최근 댓글 증가순 / 최근 스크랩 증가순 / 최근 조회 증가순 정렬

### 사용자 편의 기능
- 스크랩(북마크) 기능
- 공모전별 채팅방 (지원자 간 실시간 소통)
- 담당자 Q&A (인증된 담당자 직접 답변)
- 팀원 모집 게시판
- 수상혜택으로 채용 가산점 필터링 가능 (취준 연계 강점)

### 데이터 품질
- 실시간 공모전 공고 약 350개+
- 인턴/채용과 통합 운영으로 공모전 데이터 단독으론 중간 수준
- 담당자 인증 공고 시스템으로 정확도 향상

### 차별화 요소
- **채용 연계 공모전 필터** (수상 시 인턴/입사 가산점 여부 분류)
- **공고 달력의 완성도** (월간/주간 뷰, 내 스크랩 달력 연동)
- 지원자 채팅방으로 커뮤니티 형성

## 2. 씽굿 (thinkcontest.com)

### 핵심 기능
- 국내 최장수 공모전 전문 포털 (씽굿 + 씽유 브랜드)
- 공모전 / 대외활동 / 교육·강연 3개 영역 통합
- 씽굿Lab: 수상작 갤러리, 가이드북, 전략&통계 (자체 콘텐츠)
- 커뮤니티: 자유게시판 / 씽굿광장 / 지식Q&A / 팀원구하기

### UI/UX 특징
- 반응형 내비게이션 (햄버거 메뉴)
- 배너/슬라이더 방식 공모전 노출
- "씽굿 PICK" 추천 섹션, "지금 딱 좋은 활동" 큐레이션
- SNS 연동 (페이스북/인스타/블로그/유튜브)
- 1페이지 5개 리스트 표시 (밀도 낮음)

### 필터/검색/알림
- 분야 / 주최기관(학교·공공·대기업·중소·학회·재단) / 응모대상 / 시상금액 필터
- D-day 기준 정렬 (마감임박/여유순)
- **캘린더 뷰**: 날짜별 공모전 현황 확인 가능 (마감일 기반)
- 뉴스레터 구독 알림 (이메일)
- "방금 본 프로젝트" 열람 이력

### 사용자 편의 기능
- 주간/월간 조회수 베스트 10 실시간
- 팀원구하기 섹션 (실시간 표시)
- 공모전 전략/통계/트렌드 콘텐츠 (입문자 교육용)
- 수상자 인터뷰, 주최사 인터뷰
- 뉴스레터 (매주 새 프로젝트 안내)

### 데이터 품질
- 국내 최다 공모전 데이터 보유 (오랜 역사)
- 분야별 분류 우수
- 2026년 공모전 트렌드 자체 빅데이터 분석 제공 (독보적)

### 차별화 요소
- **공모전 콘텐츠 깊이** (전략/통계/트렌드/가이드북)
- **캘린더 뷰** 제공 (날짜별 현황)
- 오랜 신뢰도 기반 대규모 데이터베이스
- 교육·강연 카테고리 포함

## 3. 위비티 (wevity.com)

### 핵심 기능
- 공모전/이벤트/대외활동 통합 플랫폼
- 17개 분야 카테고리 (가장 세분화됨)
- 경품 이벤트 참여 섹션 별도 운영
- 공모전 아이디어&전략 자료실 (수상작 링크)

### UI/UX 특징
- 카드형 리스트 (썸네일 + 제목 + 복수 카테고리 태그 + 주최사 + 상태 배지 + 조회수)
- D-day 표시 직관적 ("D-15 접수중" 배지)
- SPECIAL / IDEA 배지로 추천 공모전 강조
- 배경 그래픽 배너 광고 영역 (수익 모델 노출)
- 페이지네이션 (10개 단위)
- 오래된 느낌의 UI (레거시 디자인)

### 필터/검색/알림
- 5중 필터: 분야(17개) / 응시대상자(일반인·대학생·청소년·어린이) / 주최사(정부·공기업·대기업·중소기업) / 시상내역(상금 규모별) / 상태(전체·스페셜·신규·마감임박·접수중·접수예정·마감)
- 이벤트 섹션 추가 필터: 경품 종류 / 응모 형태(15개) / 응모 대상
- 마감 카운트다운 D-day 표시
- 이메일 메일링 신청 서비스
- 조회수 표시로 인기도 파악

### 사용자 편의 기능
- 공모전 아이디어&전략 섹션 (수상 사례 참고)
- 분야별 17개 카테고리로 세밀한 탐색
- FAQ 및 이용약관 명시
- 제휴문의 섹션

### 데이터 품질
- 공모전 공고 수 업계 최다 수준
- 대외활동, 인턴십까지 포함
- 마감임박/신규 분류로 탐색 용이

### 차별화 요소
- **17개 세분화 분야 카테고리** (업계 최다)
- **응시대상 연령층 세분화** (어린이~일반인)
- **이벤트 참여 섹션** 별도 운영 (경품/SNS 이벤트 포함)
- 상금 규모별 필터링 가능

## 4. 대티즌 (detizen.com)

### 핵심 기능
- 대학생 타겟 특화 공모전 포털 ("대한민국 No.1 공모전 브랜드" 슬로건)
- 공모전 / 대외활동 / 이벤트 / 대티즌 콘텐츠 구성
- 채널 섹션: 대티즌위클리 (매주 최신 공모전 뉴스레터)
- 커뮤니티: Q&A / 팀플레이(팀원 모집)

### UI/UX 특징
- 실시간 공모전·대외활동 1~10위 인기 랭킹 노출
- 14개 분야 카테고리로 공모전 구분
- 이벤트 섹션 (공연/시사회 포함 — 공모전 외 문화 이벤트)
- 최근 UX에 단점 발생 (2024년 기준 사용자 지적)

### 필터/검색/알림
- 분야별 14개 카테고리 필터
- 실시간 공모전 추천 및 랭킹
- 대티즌위클리 이메일 뉴스레터 (매주)
- 접수 상태별 분류 (접수중/마감임박/접수예정/마감)

### 사용자 편의 기능
- 실시간 인기 공모전 TOP10 (1위~10위)
- 팀플레이: 팀원 모집/합류 기능
- 커뮤니티 Q&A
- 공모전 운영대행 서비스 (B2B 비즈니스 모델)
- 공모전 컨설팅 / IMC 마케팅 서비스 병행

### 데이터 품질
- 굵직한 대표 공모전 정보 빠짐없이 등록 (신뢰도 높음)
- 실시간 업데이트 강조
- 주최사 공고 직접 연결

### 차별화 요소
- **대학생 타겟 특화** 포지셔닝
- **공연/시사회 이벤트** 포함 (공모전 외 문화생활 연계)
- **공모전 운영대행 B2B** 서비스 (주최사 고객)
- 실시간 인기 랭킹 TOP10 강조

### 핵심 기능

- 별도 특화 기능 확인 불가 (소규모 플랫폼)

### 확인된 특징
- 검색 결과 및 직접 접근 모두에서 상세 정보 확인 어려움
- 공모전 포털 비교 분석 자료에서 Top5에 미포함
- youthmango.com 도메인 기반의 서비스 — 청년 대상 기회 정보 아카이브로 추정

## 우선 개선사항 TOP 15 (우선순위순)

### [P1] 즉각 반영 필수 — UX 기본기

**#1. D-day 카운트다운 배지 (마감 임박 강조)**
- 모든 경쟁사 공통 보유 기능. 카드에 "D-3", "D-day", "마감임박" 배지 없으면 이탈률 급증.
- OpportuMap 현황: 마감일 표시 있으나 D-day 배지 스타일 미확인. 직관적 강조 필요.

**#2. 스크랩/북마크 기능**
- 전 경쟁사 보유. 로그인 사용자가 관심 공모전을 저장하는 핵심 재방문 유도 장치.
- 스크랩 달력(링커리어처럼 내가 스크랩한 공모전만 캘린더에 표시) 연동 시 강력한 차별화.

**#3. 마감 알림 기능 (이메일 또는 웹 푸시)**
- 링커리어: D-1 앱 푸시. 씽굿/위비티: 이메일 뉴스레터. 대티즌: 위클리 뉴스레터.
- 최소한 이메일 알림 또는 웹 푸시 알림 구현 필요. 없으면 재방문 동기 없음.

**#4. 분야 카테고리 세분화 (최소 10개 이상)**
- 위비티 17개, 대티즌 14개. OpportuMap 현재 카테고리 공모전/해커톤/예창패/지원금은 타입 분류이지 분야 분류가 아님.
- 분야 필터 (디자인/IT/마케팅/기획/영상/문학 등) 추가 필수.

**#5. 상금/지원금 규모 필터**
- 위비티(5천만원 이상 등), 링커리어, 씽굿 모두 보유.
- 지원금 카테고리 보유한 OpportuMap이 오히려 금액 필터 없으면 역설적 UX 결함.

### [P2] 데이터 품질 개선

**#6. 응모 대상 세분화 (연령/직군)**
- 위비티: 일반인/대학생/청소년/어린이 4단계. 링커리어: 취준생 특화.
- OpportuMap: 예창패/지원금 대상이 다양 (예비창업자, 초기창업자, 청년 등). 대상자 필터 세분화 필요.

**#7. 주최기관 필터 (정부/공기업/대기업/중소기업/지자체)**
- 씽굿, 위비티, 링커리어 공통 보유.
- 정부 지원사업 중심인 예창패/지원금 카테고리 특성상 주최기관 필터가 특히 중요.

**#8. 실시간 인기/조회수 랭킹**
- 대티즌 TOP10 랭킹, 씽굿 주간/월간 조회수, 위비티 조회수 표시.
- 신규 방문자의 탐색 부담 감소. "지금 핫한 공모전" 섹션 신설 권장.

**#9. 접수 상태 배지 세분화**
